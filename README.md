#Zootopia API

Dieses Programm ist dazu da, um die Tiere die von dir gesucht werden auf einer Seite darzustellen.

##Installation

Um dieses Projekt zu installieren, klone einfach das Repository und installiere die Abhängigkeiten aus der requirements.txt mit "pip".

##Nutzung

Um dieses Programm zu starten, führe folgenden Befehl aus:
"animals_web_generator.py"

Viel Spaß! :)
